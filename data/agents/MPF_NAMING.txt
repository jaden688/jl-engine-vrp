MPF = MODULAR PERSONALITY FRAME
=================================

The word "persona" is BANNED in this project. Do not use it. Ever.

CORRECT terminology:
  - MPF               → the format/system (Modular Personality Frame)
  - MPF operator file → a single agent JSON in this folder
  - MPF character frame → what an agent's identity/voice is
  - operator identity → the loaded active agent
  - MPFProfile        → the registry struct (Modular Personality Frame Profile)

WRONG terminology:
  - "persona"         → BANNED
  - "personality profile" (as a format name) → use "MPF operator frame"
  - "Multi-Persona Format" → DEAD name, replaced by Modular Personality Frame

This applies to:
  - All JSON files in this folder (do not add "persona" to any field or note)
  - All .jl source files in src/ and BYTE/src/
  - All web copy in web/components/
  - All markdown docs
  - All comments, variable names, and string literals

If you're writing a new agent file, refer to its identity block, not its "persona".
