# Bughunt Pack

Quick commands:

```powershell
python scripts/@py-scripts/bughunt/triage_logs.py
python scripts/@py-scripts/bughunt/smoke_matrix.py
python scripts/@py-scripts/bughunt/repro_runner.py
python scripts/@py-scripts/bughunt/clutter_guard.py
```

Reports land in `runtime/dropzone/reports`.
Clutter sweep lands in `runtime/dropzone/outbox`.
